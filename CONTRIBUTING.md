# Contributing to this fork

Thank you for contributing! This repo is a fork used for experiments and fixes.
Please follow these guidelines to keep commits and PRs high quality.

- Use Conventional Commits for commit messages (`type(scope): description`).
- Create small, focused PRs. Include tests where applicable.
- Run `black` and `ruff` (if available) before creating PRs.

PR template and issue referencing:
- Mention `Fixes: #<issue>` when the PR resolves an issue.

Maintainers will review and merge following standard open-source practices.
